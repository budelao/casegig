- Cliente com saldo insuficiente → rejeição

---

## ✔️ Resgate

- Resgate com cotas suficientes → sucesso
- Resgate com cotas insuficientes → rejeição
- Violação de permanência mínima → rejeição

---

## ✔️ Fundo

- Fundo ABERTO → permite operação
- Fundo FECHADO → rejeita operação

## ✔️ Cut-off
